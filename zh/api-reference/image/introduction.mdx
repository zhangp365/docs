## 概述

TensorAI 图片 API 允许您将强大的图片生成能力集成到您的应用程序中。

## 基础 URL

```bash
https://api.tensorai.tensorslab.com/v1
```

## 身份认证

所有请求必须在 `Authorization` 请求头中包含您的 API 密钥：

```bash
Authorization: Bearer <YOUR_API_KEY>
```

## 端点

图片 API 提供以下主要功能：
- **文生图**：从文字描述生成图片
- **图生图**：基于提示词修改现有图片

查看侧边栏获取详细的端点文档。
