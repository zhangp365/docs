## 概述

TensorAI 视频 API 支持使用 Seedance 和 Veo3 等最先进模型进行程序化视频生成。

## 基础 URL

```bash
https://api.tensorai.tensorslab.com/v1
```

## 身份认证

所有请求必须在 `Authorization` 请求头中包含您的 API 密钥：

```bash
Authorization: Bearer <YOUR_API_KEY>
```

## 端点

视频 API 支持：
- **文生视频**：从文字提示词创建视频
- **图生视频**：将静态图片转化为动态视频

查看侧边栏获取详细的端点文档。
