## 身份认证

所有 API 端点需要使用 Bearer 令牌进行身份验证。在请求头中包含您的 API 密钥：

```json
"Authorization": "Bearer YOUR_API_KEY"
```

<Note>
  您可以从 [TensorAI 控制台](https://tensorai.tensorslab.com/) 生成 API 密钥。
</Note>

## 速率限制

API 使用基于积分计费。每个模型有不同的积分消耗率：

| 模型 | 类型 | 每次生成消耗积分 |
|------|------|-----------------|
| Nano Banana | 图片 | 5 积分 |
| Seedream4 | 图片 | 10 积分 |
| Seedance | 视频 (5秒) | 50 积分 |
| Seedance | 视频 (10秒) | 100 积分 |
| Veo3 | 视频 (5秒) | 80 积分 |
| Veo3 | 视频 (10秒) | 160 积分 |
